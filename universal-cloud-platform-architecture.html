5. Features

GitOps — ArgoCD

  • App-of-Apps pattern, waves 0–3
  • Self-managed ArgoCD (prune: false)
  • Multi-source Helm + values ref
  • ServerSideApply for large CRDs
  • selfHeal + automated prune
  • Webhook / 3-min poll sync

Policy — Kyverno

  • Baseline Pod Security Standards
  • Custom: resource limits, no default NS
  • restrict-external-ips (CVE-2020-8554)
  • require-crossplane-labels
  • PolicyExceptions for system NS
  • All policies: Audit mode initially

Networking — Cilium + Gateway API

  • eBPF dataplane, kube-proxy replacement
  • Gateway API v1.2.1 (replaces Ingress)
  • Bare-metal L2 ARP LoadBalancer IPs
  • IP pool: 192.168.32.100–110
  • HTTPRoute for service exposure
  • Supplemental RBAC fix for v1.19.2

Cloud Management — Crossplane

  • Providers: AWS, GCP, Azure, Terraform
  • DeploymentRuntimeConfig (not deprecated ControllerConfig)
  • ProviderConfigs via K8s Secrets
  • Cloudflare: pending official xpkg
  • Multi-cloud IaC from cluster

CI/CD Pipeline

  • Gitleaks: secret scanning (full history)
  • yamllint: YAML style enforcement
  • kubeconform: K8s schema validation
  • Kyverno apply: policy syntax check
  • ShellCheck: script analysis
  • ArgoCD diff preview on PRs
  • Renovate: automated dep updates

Services & Pending

  • SearXNG — private search, search.aos.local
  • Pending: Observability (Prometheus + Loki)
  • Pending: Cloud credentials setup
  • Pending: Cloudflare provider
  • Pending: TLS on platform-gateway
  • Pending: ArgoCD SSO / OIDC

6. Deployment & Operations

Day-0 Bootstrap (run once)

  • bootstrap/k8s/prep-node.sh — OS prep on each NUC
  • bootstrap/k8s/init-control-plane.sh — kubeadm init aos01
  • bootstrap/k8s/join-control-plane.sh — join aos02/03
  • bootstrap/bootstrap-argocd.sh — Helm install ArgoCD + apply root app
  • ArgoCD reconciles all waves automatically

Day-2 Operations

  • All changes via PR → merge → ArgoCD sync
  • Renovate opens dependency PRs every weekend
  • ArgoCD diff preview on every manifest PR
  • scripts/setup-credentials.sh — cloud provider secrets
  • scripts/setup-searxng-secret.sh — SearXNG key rotation
